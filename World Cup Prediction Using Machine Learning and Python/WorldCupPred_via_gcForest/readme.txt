基于GCForest的2022年世界杯结果预测模型

该项目是我python课程的个人大作业，并与其它几位组员的世界杯数据分析项目共同组成了小组大作业。

即我们小组8名同学每人完成了一份个人大作业，而作为小组也在个人作业的基础上进一步分析，完成了一份小组大作业，共计9份作业。个人作业将分别提交在每个人的“个人/小组大作业中”，小组大作业提交在我的“个人/小组大作业中”和“小组presentation提交"中。

该项目主要逻辑请见目录中大作业报告幻灯片。项目文件主要结构如下：

examples 文件夹下是项目主要内容，包括数据处理源代码data_process.py, 机器学习源代码pred.py，和一些数据文件。

lib 文件夹包含gcforest的主要内容，详见参考文献[1]。

requirement_for_GCForest.txt包含项目所需package，如要运行请确保正确安装了它们以及其他import涉及的package.
